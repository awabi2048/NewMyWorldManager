# `/myworld` 一覧画面契約

## 参照元

MWM-Chanpon の `ChanponPlayerWorldMenuProvider` と、2026-07-11 の7列共通レイアウト変更を参照元とする。

## 維持する仕様

- CC-System の `sevenColumnPage` が返す寸法、一覧スロット、ページ移動、戻る位置を使用する。
- 上下の標準枠は黒色、一覧領域を含む未使用スロットは灰色の装飾で埋める。
- フッター中央を基準に、作成を左2スロット、プレイヤー情報を中央、個人設定を右2スロットへ配置する。
- ワールドアイコンの情報と、左クリック、右クリック、Shift+左クリックの操作はMWMの機能として維持する。
- 未処理の招待・申請がある場合は、中央のプレイヤー情報へ件数、最新日時、一覧を開く操作を追加する。

## MWM-Chanponとの差分

- 右クリックによる簡単作成はChanpon固有機能のため、MWM基盤には追加しない。
- ワールドポイント、スロット、招待・申請はMWM基盤の状態であるため、中央のプレイヤー情報へ表示する。

## 変更理由

7列共通レイアウトへの移行後、一覧行の左右端が背景で埋まらず、招待・申請の独立ボタンによって3点配置も崩れていた。共通描画APIによる空き領域の充填と、中央情報への申請導線統合によって参照元の画面構造へ戻す。
